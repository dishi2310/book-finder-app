# 📚 Book Finder Web App

## 🔰 Project Description

The Book Finder Web App is a simple and interactive web application that allows users to search for books and view their details such as title, author, and available formats.

Users can also open and read books directly in their browser.

---

## 🚀 Features

* 🔍 Search for books using keywords
* 📚 Display book results dynamically
* 📖 Open and read books in a new tab
* ⬆️ Sort books by publish year
* ❤️ Add books to favorites (using localStorage)
* 🌙 Dark mode toggle

---

## 🌐 API Used

This project uses the **Gutendex API**, which provides free access to thousands of public domain books.

API Endpoint:
https://gutendex.com/books

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation, Fetch API)

## ⚙️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/book-finder-app.git
   ```

2. Open the project folder

3. Open `index.html` in your browser
